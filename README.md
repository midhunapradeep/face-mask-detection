# Face Mask Detection using MobileNetV2

## Overview

This project detects whether a person is wearing a face mask in real time using a webcam.

## Technologies Used

* Python
* TensorFlow
* Keras
* OpenCV
* MobileNetV2
* NumPy

## Features

* Real-time webcam detection
* Face detection using Haar Cascade
* Mask / No Mask classification
* Lightweight deep learning model

## Installation

Install the required libraries:

pip install -r requirements.txt

## Run the Project

python detect_mask.py

## Project Workflow

Webcam → Face Detection → Image Preprocessing → MobileNetV2 → Prediction → Display Result
